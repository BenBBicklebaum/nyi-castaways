Methodology

SOS (Strength of Schedule) — Average points percentage of remaining opponents, calculated live from current standings filtered to games not yet played.

Projected Points — Current pts ÷ GP × 82, rounded to nearest integer.

Clinch Thresholds — Based on pace projections of the first team outside each cutline. Requires NYI to reach that threshold to clinch on pace.

Deep Insights — AI-generated each page load using current standings, SOS, H2H schedule, and convergence data. Powered by Claude.

Rooting Interest — Multi-factor threat score: pts gap, projected pts, SOS rank, games remaining, H2H vs NYI, WC position proximity.
